## Description
Ce projet est une interface bancaire épurée réalisée dans le cadre d'un cursus UX Design.
L'objectif était de créer une expérience **Mobile First** élégante, inspirée par les codes  du secteur bancaire institutionnel.

## Caractéristiques Techniques
* **Structure** : HTML5 Sémantique (Header, Main, Footer).
* **Style** : CSS3 utilisant la méthodologie **BEM** (Block Element Modifier).
* **Mise en page** : Utilisation de **Flexbox** pour l'alignement et le centrage.
* **Design** : Minimaliste, palette de couleurs Beige & Blanc, typographie Serif.

## Sprints Réalisés
1. **Sprint 1** : Base HTML et structure solide.
2. **Sprint 2** : Intégration CSS et mise en page Flexbox.
3. **Sprint 3** : Validation des conventions BEM et Peer-coding.
4. **Sprint 4** : Mise en ligne via GitHub Pages.
